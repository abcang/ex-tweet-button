# ExTweetButton

ページ内のTweetButtonを使ってツイートします。いちいちボタンを探す必要はありません。

## Install

[**Chrome** extension](https://chrome.google.com/webstore/detail/extweetbutton/mpcnndaommbnhccepiefejfgjdbiifke)
